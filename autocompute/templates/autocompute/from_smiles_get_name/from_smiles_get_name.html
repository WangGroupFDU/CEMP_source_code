    Batch Query Molecular Names

    The user needs to upload an EXCEL document, which should contain a 'SMILES' column. After clicking Submit, PubChem is used to query the Name corresponding to the SMILES. Illegal characters in the Name can be replaced with '_', generating a computer-safe name. After the task is submitted, the result can be queried in the Query module.
