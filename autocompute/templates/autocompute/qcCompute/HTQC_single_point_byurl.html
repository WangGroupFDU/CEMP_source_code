    1. Introduction to the High-Throughput Quantum Chemistry Calculation Module

    This module facilitates high-throughput quantum chemistry calculations based on Gaussian16 or ORCA, incorporating both structure optimization and single-point energy calculations. Users can choose between Gaussian16 and ORCA for their calculations.

    Calculation Software Options:

    • Gaussian16: A commercial software requiring a license (available only to collaborating users). The calculation results on this platform are intended for educational purposes only and cannot be used for publication or commercial purposes without purchasing a license.
    • ORCA: Free for academic users. Suitable for both main group elements and transition metal-related calculations.

    Important Note on Software Usage:

    • Calculation results from Gaussian16 and ORCA cannot be mixed. Users must choose one software for all calculations in a project.

    Structure Optimization:

    • For Gaussian16:
      • Theoretical level: B3LYP/6-311G**
      • Dispersion correction: GD3BJ
      • Frequency correction factor: 0.9682
    • For ORCA:
      • Theoretical level: B3LYP/def2-TZVP
      • Acceleration method: RIJCOSX
      • Dispersion correction: GD3BJ

    The program automatically checks for optimization failures or imaginary frequencies and performs error correction if necessary.

    Single-Point Energy Calculation:

    • For Gaussian16:
      • Theoretical level: M062X/6-311+(2d,p)
      • Dispersion correction: GD3
    • For ORCA:
      • Theoretical level: wB97M-V/def2-TZVP
      • Acceleration method: RIJCOSX

    The program automatically checks for calculation failures and performs error correction if necessary.

    Users need to download the provided Excel file, fill in the calculation parameters, and upload the completed file. The program will automatically execute the calculations and download the results upon completion.

    Included Calculation Results:

    • Name
    • SMILES
    • Single-point energy (Hartree)
    • Free energy correction (Hartree)
    • Enthalpy correction (Hartree)
    • Entropy (J/mol K)
    • HOMO (Hartree)
    • LUMO (Hartree)
    • HOMO-LUMO gap (Hartree)
    • Dipole moment (Debye)
    • Gibbs free energy (Hartree)
    • Enthalpy (Hartree)
    • HOMO-LUMO gap


    2. Notes

    1. Software License and Usage:

    • Gaussian16: A commercial software requiring a license (available only to collaborating users). The calculation results on this platform are intended for educational purposes only and cannot be used for publication or commercial purposes without purchasing a license.
    • ORCA: Free for academic users. Suitable for both main group elements and transition metal-related calculations.
    • Calculation results from Gaussian16 and ORCA cannot be mixed. Ensure all calculations in a project use the same software.

    2. Excel File:

    • Users must use the provided Excel file to input relevant parameters.
    • Ensure the Name column does not contain illegal characters (e.g., Chinese characters, commas, periods, slashes, etc.).
    • Ensure the SMILES notation is correct (RDKit can be used to validate SMILES), otherwise the program will not function correctly.

    3. Fixed Parameters:

    • The parameters for structure optimization and single-point energy calculations are fixed in the high-throughput calculation program to prevent errors that could result in meaningless results.

    4. Applicability:

    • The calculation parameters for Gaussian16 are only suitable for organic molecular systems.
    • For ORCA, the parameters support both main group elements and transition metal-related calculations.
    • Ensure the system does not contain transition metals when using Gaussian16, as this could lead to erroneous results. For transition metals, use ORCA.
