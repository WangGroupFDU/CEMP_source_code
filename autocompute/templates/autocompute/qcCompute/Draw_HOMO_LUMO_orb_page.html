    This module computes HOMO-LUMO orbital figures from user-supplied Gaussian checkpoint (.chk/.fchk) or ORCA check point (.gbw/.molden)files. Upon completion, it outputs volumetric grid files (.cub), corresponding formatted checkpoint files (.fchk, for Gaussian; .molden, for ORCA), and VMD-rendered images (.tga). Note: Only checkpoint files produced by Gaussian or ORCA calculations are accepted as valid input.
