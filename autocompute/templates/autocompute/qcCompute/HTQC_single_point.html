    1. Introduction to the High-Throughput Quantum Chemistry Calculation Module

    This module facilitates high-throughput quantum chemistry calculations based on Gaussian16, incorporating both structure optimization and single-point energy calculations. Structure Optimization: • Theoretical level: B3LYP/6-311G** • Dispersion correction: GD3BJ • Frequency correction factor: 0.9682 • The program automatically checks for optimization failures or imaginary frequencies and performs error correction if necessary. • Single-Point Energy Calculation: • Theoretical level: M062X/6-311+(2d,p) • Dispersion correction: GD3 • The program automatically checks for calculation failures and performs error correction if necessary. Users need to download the provided Excel file, fill in the calculation parameters, and upload the completed file. The program will automatically execute the calculations and download the results upon completion. Included Calculation Results: • Name • SMILES • Single-point energy (Hartree) • Free energy correction (Hartree) • Enthalpy correction (Hartree) • Entropy (J/mol K) • HOMO (Hartree) • LUMO (Hartree) • HOMO-LUMO gap (Hartree) • Dipole moment (Debye) • Gibbs free energy (Hartree) • Enthalpy (Hartree) • Electrochemical window (estimated using the HOMO-LUMO method)



    2. Notes

    1. Gaussian16 License: • Gaussian16 is a commercial software. The calculation results on this platform are intended for educational purposes only and cannot be used for publication or commercial purposes without purchasing a license. 2. Excel File: • Users must use the provided Excel file to input relevant parameters. • Ensure the Name column does not contain illegal characters (e.g., Chinese characters, commas, periods, slashes, etc.). • Ensure the SMILES notation is correct (RDKit can be used to validate SMILES), otherwise the program will not function correctly. 3. Fixed Parameters: • The parameters for structure optimization and single-point energy calculations are fixed in the high-throughput calculation program to prevent errors that could result in meaningless results. 4. Applicability: • The current calculation parameters are only suitable for organic molecular systems. • Ensure the system does not contain transition metals, as this could lead to erroneous results.

    5. !!!!! Important Considerations !!!!!: • After the user uploads the HTQC.xlsx file and clicks the Start button, quantum chemistry calculations will begin. The computation time will increase with the number of molecules, the number of atoms within the molecules, and the complexity of the molecules. • Once the Start button is clicked, the user should not exit or refresh the current page, as this may result in the loss of calculation results. Please do not close or refresh the browser until the computation is complete. • Quantum chemistry calculations are relatively time-consuming tasks. Users are advised to be patient; as long as there are no red error messages on the browser page, the process is running normally. Please do not close the browser to avoid calculation failure or other issues. • Due to current computational resource limitations, the maximum level of parallelism provided by this program is 2 (i.e., only 2 quantum chemistry tasks can be executed simultaneously).
