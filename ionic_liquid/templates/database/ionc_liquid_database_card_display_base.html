Users can explore three curated databases by selecting the cards above. Each database provides access to comprehensive IL-related data.

  • Cation Database: Comprises quantum‐chemical properties for 3,774 distinct cation species.
  • Anion Database: Comprises quantum‐chemical properties for 2,220 distinct anion species.
  • Generated IL Database: Contains machine‐learning–predicted conductivity, melting point, and electrochemical window for 100,000 in silico–generated ionic liquids.
  • Experimental IL Database: Contains experimentally measured or quantum‐chemistry–computed conductivity, melting point, and electrochemical window for approximately 3,600 real ionic liquids.
