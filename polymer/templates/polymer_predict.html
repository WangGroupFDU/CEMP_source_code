1. Introduction to the Polymer Property Prediction Models

Using XGBoost, separate predictive models were developed for five different properties of polymers: Glass transition temperature, static dielectric constant, melting point, tensile strength and Young's modulus. The specific training processes for these models follow the methodology described in the published paper ‘Wang J F, Sun Y B, Chen Q T, et al. OpenPoly: A Polymer Database Empowering Benchmarking and Multi-property Predictions[J]. Chinese Journal of Polymer Science, 2025: 1-12.’, which includes detailed training procedures and model performance.

2. Notes

• Users must use the provided Excel file to input relevant parameters.
• Ensure the Name column does not contain illegal characters (e.g., Chinese characters, commas, periods, slashes, etc.).
• Ensure the PSMILES notation is correct (RDKit can be used to validate SMILES), otherwise the program will not function correctly.
